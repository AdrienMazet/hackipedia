# Installation
```
cd hackipedia-chrome
npm install
npm run build
````

Dans votre naviguateur (exemple avec Chrome) :
- aller sur [chrome://extensions/](chrome://extensions/)
- activer le "Developer mode" (bouton en haut à droite)
- cliquez sur "load unpacked"
- sélectionnez le répertoire "dist" issu du build
- allez dans les paramètres de l'extension. Entrez votre clé d'API mistral (premier champ) et ElevenLabs (second champ)

C'est prêt, vous pouvez commencer à apprendre sur [WikipediA](https://fr.wikipedia.org/)
